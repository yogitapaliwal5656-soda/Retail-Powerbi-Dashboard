Project Overview

This project presents a Retail Sales Analysis Dashboard built using Power BI.
The dashboard helps analyze customer behavior, product performance, and sales trends across different locations and customer segments.

Dataset Information

Total Rows: 300,845

Total Columns: 25

Data Type: Retail Sales Data

Tools Used

Power BI

Data Modeling (Star Schema)

DAX Measures

Data Cleaning & Transformation (Power Query)

Key Insights

Sales performance by City and Country

Customer segmentation analysis (Regular, Premium, New)

Product category and brand performance

Rating and customer behavior analysis

Data Model

The project uses a Star Schema Model with:

Fact Table: retail_data

Dimension Tables: Segment, City-Country, Product

Measures Table for DAX calculations

Dashboard Features

Interactive filters and slicers

KPI cards for important metrics

Visual analysis for sales and customer segments
